About owncloud

OwnCloud is free and open source software
that operates as a very simple way to set
up your own syncing, Dropbox-like cloud
storage system on your own server or web site.
With owncloud, manipulating, configuring
and accesing of appplications online is made
easier. It allows accessing of applications as
utilities, over the internet.
